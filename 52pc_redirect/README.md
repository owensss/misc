52pcgame_linkFix
================

chrome & firefox plugin to fix 52pcgame broken links.
用于修复整合贴内帖子无法访问的 chrome 插件

##### 用法

###### chrome 用户
1. 下载 52pc_linkfix.crx.
2. 拖到 chrome 上 - -b

###### firefox 用户
1. 下载 52pc_linkfix.xpi
2. 拖到 firefox 上 - -b

###### 原理
把已经挂掉的 bbs.52pcgame.com 的正则替换成现在的 http://121.199.23.180/ 网址

###### Note
只处理了 a.href...
